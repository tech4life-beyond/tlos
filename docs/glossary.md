# Glossary (Canonical Terms)

This glossary defines canonical terms used across the Tech4Life ecosystem.

## Canonical terms
- **TLOS Governance Standard**: The normative governance/process doctrine. (Not product runtime source of truth.)
- **Product Creation Lifecycle (PCL)**: The single canonical stage model for product progression (9 stages).
- **Factory Profile**: An implementation mode of the PCL (e.g., hardware licensing-ready, platform foundation).
- **Product Pack**: Stage-bounded artifact bundle for one product, governed by metadata contract.
- **Product Registry Record**: Canonical machine-readable product entry with schema version.
- **TOIL Legal Baseline**: The current pinned legal baseline (license + templates + amendment log).
- **Release Gate**: Composite pass/fail policy across legal, ethics, technical, registry, and finance controls.
- **Contributor Role**: Observer / Contributor / Creator / Architect.
- **Capability Grant**: Permission granted by role + curriculum completion + governance approval.
- **Normative vs Informative**: Every policy statement must be classed to avoid drift.
