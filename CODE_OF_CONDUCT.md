# Code of Conduct

Tech4Life & Beyond exists to create technology with integrity, accountability, and positive impact.
Participation in this ecosystem requires respectful, safety-minded collaboration.

## Our standards

We commit to:
- Treat others with respect and professionalism.
- Assume good intent, seek clarity, and resolve disagreements with evidence and calm.
- Prioritize safety, ethics, and legality in all discussions and contributions.
- Protect confidential information in private repositories and private channels.

Unacceptable behavior includes:
- Harassment, discrimination, or threats
- Doxxing or sharing private data without consent
- Encouraging unsafe or illegal actions
- Bad-faith disruption, trolling, or repeated refusal to follow governance rules

## Scope

This code of conduct applies to:
- GitHub repositories, issues, PRs, and discussions
- Community channels and events associated with Tech4Life & Beyond

## Enforcement

If a violation occurs, maintainers may take action including:
- Requesting changes
- Removing content
- Temporarily or permanently restricting participation

## Reporting

If you experience or witness misconduct:
- Prefer private reporting to a repository owner/maintainer.
- If the report is security-related, follow `SECURITY.md`.

## Governance alignment

This document is aligned with TLOS governance and TOIL integrity principles.
