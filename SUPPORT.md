# Support

This repository contains the Tech4Life & Beyond documentation and standard definitions for TLOS.

## What we support here
- Documentation structure and consistency
- Link integrity and anchor validation
- Formatting and front matter validation rules

## How to get help
1. Open a GitHub Issue with:
   - A clear title describing the problem
   - The affected file path(s)
   - A copy of the CI error output (logs)
   - Steps to reproduce locally, if applicable

## Confidentiality
Do not include private/legal/finance details in issues.

## Maintainers
Changes to core documents may require CODEOWNERS review.
